pyConTextKit Views¶

This module contains the views that are used in the pyConTextKit application.

pyConTextKit.views.complete(request)¶

This page is rendered by the run view when the Annotate feature is finished.

pyConTextKit.views.index(request)¶

This in the index page.

pyConTextKit.views.itemData_edit(request, itemData_id=None)¶

This method takes an itemDatum ID as an argument and renders a form for the user to edit the specified extraction criterion. If no argument is supplied, a blank form is rendered, which the user can use to enter a new criterion.

Note: A useful addition to this would be examples that the user can use as a reference.

pyConTextKit.views.itemData_filter(request, supercat)¶

This method takes a supercategory name as an argument and renders a view of the criteria in this supercategory.

pyConTextKit.views.itemData_view(request)¶

This provides an overview of all the extraction criteria. It also provides a search box that the user can use to search for relevant search terms.

pyConTextKit.views.logout_view(request)¶

This logs the user out of the application.

pyConTextKit.views.run(request)¶

This executes the Annotate feature.
