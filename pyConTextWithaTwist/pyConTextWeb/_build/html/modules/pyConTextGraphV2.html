pyConTextKit pyConTextGraphV2¶

This module contains three class definitions that are used in the pyConText algorithm. The pyConText algorithm relies on regular expressions to identify sub-texts of interest

  1. termObject: a class that describes terms of interest within the text
  2. tagObject: a class inherited from termObject that describes modifiers
  3. pyConText: a class that implements the context algorithm
